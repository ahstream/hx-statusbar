# README

## Develop & Test

```
// Will open http://localhost:8080/ with test page using src/example.js

yarn dev
```

## Create NPM package

```
yarn publish

// From scratch

yarn build
npm login
npm publish --access public

//
```

## Consume NPM package

```
// Add from NPM
npm add @ahstream/hx-statusbar

// Add local
npm add ../hx-statusbar
```
